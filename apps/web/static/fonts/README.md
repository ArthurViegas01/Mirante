# Fontes self-hosted

`app.css` referencia estes arquivos via `@font-face`. Coloque os `.woff2` aqui
(servidos em `/fonts/...`). Até que existam, o app usa as fallback stacks
(system sans/serif/mono) — o layout não quebra.

Arquivos esperados:

| Arquivo | Família | Peso/estilo |
|---|---|---|
| `figtree-variable.woff2` | Figtree | 300–900, normal (variável) |
| `instrument-serif-regular.woff2` | Instrument Serif | 400, normal |
| `instrument-serif-italic.woff2` | Instrument Serif | 400, itálico |
| `geist-mono-variable.woff2` | Geist Mono | 400–600, normal (variável) |

Fontes: Figtree e Geist Mono (Google Fonts / repositórios oficiais, OFL),
Instrument Serif (Google Fonts, OFL). Baixe os `.woff2` e versione-os aqui na F4
(polish), ou antes se quiser a tipografia final já no F0.
